Problem 8-2 — Gravity station 計算

題目 (摘錄)

Given the following data for a gravity station:

  • Latitude: 48.1195° N
  • Longitude: 12.1878° E
  • Elevation: 487.9 m
  • Observed gravity: 980,717.39 mGal

a) Calculate the: i) theoretical gravity; ii) free-air correction; iii) Bouguer correction; iv) free-air gravity anomaly; and v) Bouguer gravity anomaly.

b) Suppose the station had exactly the same latitude, longitude and observed gravity, but it was located on the surface of the ocean, over a water depth of 487.9 m. Recompute the free-air and Bouguer gravity anomalies.

Python 程式 (可下載並執行)

說明：此程式使用常見的重力修正公式（GRS80 準則近似式、自由空氣修正和簡單 Bouguer slab 修正）。密度預設岩石 2670 kg/m³；海水情況以 1000 kg/m³ 處理。

#!/usr/bin/env python3
"""
Gravity corrections for Problem 8-2
Outputs: theoretical gravity (mGal), free-air correction (mGal),
Bouguer correction (mGal), free-air anomaly (mGal), Bouguer anomaly (mGal)
"""
import math

def theoretical_gravity(lat_deg):
    # GRS80 normal gravity formula (Somigliana-like approximation)
    phi = math.radians(lat_deg)
    g0 = 978032.677  # mGal, nominal for GRS80 (value depends on unit choices)
    # Somigliana coefficients (approximate)
    g = g0 * (1 + 0.0053024 * math.sin(phi)**2 - 0.0000058 * math.sin(2*phi)**2)
    return g


def free_air_correction(h_m):
    # standard free-air gradient ~ 0.3086 mGal per meter
    return 0.3086 * h_m


def bouguer_correction(h_m, density_kg_m3=2670):
    # Simple Bouguer slab: 2 * pi * G * rho * h, expressed in mGal
    # Factor 0.04193 approximates 2*pi*G in units converting to mGal when rho in g/cm3
    # More common simplified formula: 0.04193 * rho(g/cm3) * h(m)
    rho_g_cm3 = density_kg_m3 / 1000.0
    return 0.04193 * rho_g_cm3 * h_m


def compute_all(lat_deg, h_m, g_obs_mgal, rock_density=2670):
    g_th = theoretical_gravity(lat_deg)
    FAC = free_air_correction(h_m)
    BC_rock = bouguer_correction(h_m, rock_density)
    # anomalies: observed + FAC - theoretical  (free-air anomaly)
    free_air_anomaly = g_obs_mgal + FAC - g_th
    bouguer_anomaly = g_obs_mgal + FAC - BC_rock - g_th
    return {
        'g_theoretical': g_th,
        'free_air_correction': FAC,
        'bouguer_correction_rock': BC_rock,
        'free_air_anomaly': free_air_anomaly,
        'bouguer_anomaly_rock': bouguer_anomaly
    }

if __name__ == '__main__':
    lat = 48.1195
    h = 487.9
    g_obs = 980717.39
    results_land = compute_all(lat, h, g_obs, rock_density=2670)
    results_ocean = compute_all(lat, h, g_obs, rock_density=1000)  # water density

    print('Results (land station):')
    for k,v in results_land.items():
        print(f'{k:30s}: {v:12.4f}')

    print('\nResults (station over water, water density used in Bouguer):')
    for k,v in results_ocean.items():
        print(f'{k:30s}: {v:12.4f}')
