8-2 重力測站校正與異常計算

已知資料

  • 緯度 Latitude：48.1195° N
  • 經度 Longitude：12.1878° E（本題不使用）
  • 海拔高度 Elevation：487.9 m
  • 實測重力 Observed gravity：980,717.39 mGal

a) 陸地測站計算（海拔 +487.9 m）

項目公式 / 說明數值
i) 理論重力 γ₀1967 國際重力公式
γ₀ = 978032.68 × (1 + 0.0053024 sin²φ − 0.0000058 sin²2φ) mGal		981,182.03 mGal
ii) 自由空氣校正 FAC−0.3086 mGal/m × h−150.62 mGal
iii) 布格校正 BC（簡單布格板）−0.1119 mGal/m × h（ρ = 2.67 g/cm³）−54.62 mGal
iv) 自由空氣重力異常 FAAgobs − γ₀ − FAC−314.02 mGal
v) 布格重力異常 BAFAA − BC 或 gobs − γ₀ − FAC − BC−259.40 mGal

b) 假設測站位於海洋表面下方 487.9 m（水深 487.9 m）

此時高度 h = −487.9 m，且布格校正需考慮海水替換岩石的密度差。

項目說明數值
自由空氣校正 FAC−0.3086 × (−487.9)+150.62 mGal
布格校正 BC（海洋）密度差 Δρ = 2.67 − 1.03 = 1.64 g/cm³
係數 ≈ −0.0689 mGal/m × 487.9 m		−33.62 mGal
自由空氣異常 FAAgobs − γ₀ − FAC+685.98 mGal
布格異常 BAFAA − BC（海洋）+652.36 mGal

完整 Python 程式（可直接執行）

import math

# 已知資料
lat_deg = 48.1195
h_land = 487.9
g_obs = 980717.39    # mGal

# 轉弧度
lat = math.radians(lat_deg)
sin2 = math.sin(lat)**2
sin4 = sin2**2

# i) 理論重力 (1967 公式)
gamma0 = 978032.68 * (1 + 0.0053024*sin2 - 0.0000058*sin4)

print("=== a) 陸地測站 (海拔 487.9 m) ===")
FAC_land = -0.3086 * h_land
BC_land  = -0.1119 * h_land
FAA_land = g_obs - gamma0 - FAC_land
BA_land  = FAA_land - BC_land

print(f"理論重力 γ₀       = {gamma0:.3f} mGal")
print(f"自由空氣校正      = {FAC_land:+.3f} mGal")
print(f"布格校正          = {BC_land:+.3f} mGal")
print(f"自由空氣異常 FAA  = {FAA_land:.3f} mGal")
print(f"布格異常 BA       = {BA_land:.3f} mGal")

print("\n=== b) 海洋測站 (水深 487.9 m) ===")
h_ocean = -487.9
FAC_ocean = -0.3086 * h_ocean
BC_ocean  = -0.0689 * abs(h_ocean)   # 海水替換校正

FAA_ocean = g_obs - gamma0 - FAC_ocean
BA_ocean  = FAA_ocean - BC_ocean

print(f"自由空氣校正      = {FAC_ocean:+.3f} mGal")
print(f"布格校正 (海洋)   = {BC_ocean:+.3f} mGal")
print(f"自由空氣異常 FAA  = {FAA_ocean:.3f} mGal")
print(f"布格異常 BA       = {BA_ocean:.3f} mGal")
