Gravity Calculation Homework 8-2

Based on GRS80 gravity model & Bouguer corrections

Problem Description

Given a gravity station with the following properties:

  • Latitude: 48.1195° N
  • Longitude: 12.1878° E
  • Elevation: 487.9 m
  • Observed gravity: 980,717.39 mGal

Compute theoretical gravity, free-air correction, Bouguer correction, free-air anomaly, and Bouguer anomaly.

Python Code

#!/usr/bin/env python3
import math

def theoretical_gravity(lat_deg):
    phi = math.radians(lat_deg)
    g0 = 978032.677
    g = g0 * (1 + 0.0053024*math.sin(phi)**2 - 0.0000058*math.sin(2*phi)**2)
    return g

def free_air_correction(h_m):
    return 0.3086 * h_m

def bouguer_correction(h_m, density_kg_m3=2670):
    rho_g_cm3 = density_kg_m3 / 1000.0
    return 0.04193 * rho_g_cm3 * h_m

def compute_all(lat_deg, h_m, g_obs_mgal, rock_density=2670):
    g_th = theoretical_gravity(lat_deg)
    FAC = free_air_correction(h_m)
    BC = bouguer_correction(h_m, rock_density)
    free_air_anomaly = g_obs_mgal + FAC - g_th
    bouguer_anomaly = g_obs_mgal + FAC - BC - g_th
    return g_th, FAC, BC, free_air_anomaly, bouguer_anomaly

if __name__ == "__main__":
    lat = 48.1195
    h = 487.9
    g_obs = 980717.39
    print(compute_all(lat, h, g_obs))
