HW7 – Gravity Calculation

Problem 8-2

題目

Calculate the gravitational acceleration at a latitude of 45° on an oblate Earth using the Somigliana formula: 

g(φ) = gₑ * (1 + k·sin²φ) / √(1 - e²·sin²φ)
  • gₑ = 9.7803253359 m/s²
  • k = 0.00193185265241
  • e² = 0.00669437999013
  • φ = 45°

Python 程式碼

下載 Python 程式 
import math

g_e = 9.7803253359
k = 0.00193185265241
e2 = 0.00669437999013

phi_deg = 45
phi = math.radians(phi_deg)

g = g_e * (1 + k * math.sin(phi)**2) / math.sqrt(1 - e2 * math.sin(phi)**2)

print(f"Latitude = {phi_deg}°")
print(f"Gravitational Acceleration g = {g:.10f} m/s^2")

計算結果

執行結果如下：

Latitude = 45°
Gravitational Acceleration g = 9.8061992033 m/s²
